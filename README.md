## 📜프로젝트 소개
>**개발 기간 :** 2025/01/08 ~ 2025/01/17

**영화 여기 다모았다! DAMOATTA!**  
여러 영화를 검색하고, 좋아하는 영화를 보관함에 저장할 수 있는 웹 애플리케이션입니다.
현재 상영중인 영화를 실시간으로 확인할 수 있고, 실시간 검색 기능을 통해 원하는 영화를 찾아볼 수 있습니다. 
좋아요를 누른 영화는 보관함은 물론, 영화 카드에도 표시되어 어디에서나 확인할 수 있습니다.
